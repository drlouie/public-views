Intel® SR2100 2U Rack Server Chassis  
    Server Chassis Features  
The compact SR2100 server chassis delivers leading technology and large storage capacity. It is ideal for service provider, parallel server application, and other high-transaction, space-constrained computing environments. By combining the SR2100 server chassis with an Intel® Server Board STL2, powerful Intel® Pentium® III processors, and an optional redundant power supply, you can quickly and easily build a robust 2U server.
