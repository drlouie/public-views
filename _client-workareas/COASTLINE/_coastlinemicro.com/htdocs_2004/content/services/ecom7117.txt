                  <font class="stext"><br>
                  Boost speed and ramp online transaction security with the Coastline 
                  Micro 7117 e-Commerce Accelerator. Compact and easy to install 
                  between the router and server farm, the Coastline Micro 7117 
                  e-Commerce Accelerator offloads Secure Sockets Layer (SSL) encryption/decryption 
                  to improve the efficiency of web content delivery.
                  <p>In cascade mode, multiple units can be scaled to accommodate 
                    peak traffic periods for up to 3,000 new SSL connections per 
                    second. SSL traffic overflow can be transferred to the next 
                    unit or back to the server. This failover/pass-through technology 
                    provides a redundant fault-tolerant solution so e-Commerce 
                    remains quick, reliable and secure. In one-arm mode, certain 
                    layer-4 switches and load-balancers direct traffic to the 
                    Coastline Micro 7117 e-Commerce Accelerator for processing, 
                    which extends the appliance's ability to scale past 3,000 
                    new SSL connections per second.</p>
                  <p>Client certificate authentication improves security with 
                    the option of requiring a client certificate to be presented 
                    before web access is granted. The Coastline Micro 7117 e-Commerce 
                    Accelerator can restrict access further by verifying a client 
                    certificate's authenticity and validity against a Certificate 
                    Revocation List (CRL).</p>
                  <p>The unit can be managed directly with the serial connection 
                    port for high security or remotely via telnet SSH or SNMP. 
                    Event monitoring capabilities allow viewing key appliance 
                    metrics like SSL connections per second (cps), and provide 
                    event alerts such as encryption status change, refused SSL 
                    connections, threshold alerts, overload alerts and network 
                    link failure. With SNMP MIB and MIB II support the Coastline 
                    Micro 7117 e-Commerce Accelerator can be monitored using standard 
                    SNMP network management applications.<br>
                  </p></font>
