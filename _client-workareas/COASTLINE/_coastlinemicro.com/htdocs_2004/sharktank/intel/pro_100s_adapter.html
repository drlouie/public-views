Intel® Pro/100 S Desktop Adapter (RJ45)  
 
Protect sensitive data traveling on the LAN while preserving network performance with 10/100Mbps managed connections and IPSec encryption offloading. The Intel® PRO/100 S Desktop Adapter features the Intel 82550EY Fast Ethernet controller with integrated security co-processor, Intel SingleDriver™ technology for compatibility across Intel PRO/100 Network Adapters, and advanced management capabilities for easy administration.

The PRO/100 S adapter includes all of the features of the PRO/100+ Management adapter, with the addition of an Intel 82549ED IPSec encryption co-processor on board. This component frees the system processor from data encryption and authentication tasks when used with the IPSec (IP Security) protocol in the supported operating systems.
