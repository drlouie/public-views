// ProductType = Product classification for searching purposes

// Description = Descriptive information about product for use throughout the site's interface

// Keynotes = Used to present descriptive product information to the user as keynotes
// Templated HTML information stored in DB

// Keywords = As many descriptive keywords as necessary. Will be used in
// keyword searches within the store's inventory

// Reconfig = Is this system re-configurarble or is it sold as is?
// One of 2 options Y or N

// SneakPeak = Product sneak peak image, if any, if none this product is not considered for
// random sneak peaks

// SmallPhoto = Location of small photo, for use in the system specifications screen

// LargePhoto = The locations of all large photos available for this product
// Up to t0 250 char

// Weight = Weight of product in Pounds and Ounces

// SystemBase = This is the base of the system; Chassis/Power Supply/Motherboard

// Suggestion = Bases should be saved to the system as Kits and then called called from
// the configuration of a system

// ProcTypes = The types of processors this SystemBase can take, for system reconfig purposes

// Processor = Thw actual processor chosen for this system

// ProcQuantity = If necessary this count can be changed, this will not be an option for the user,
// Unless a kit's motherboard has capabilities for multiple processors
