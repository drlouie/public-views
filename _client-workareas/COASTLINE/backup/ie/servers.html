Entry Level
Zephyr T440BX
This system supports the Pentium II and/or Pentium III processors, optimized for the 100-MHz system bus. With unmatched expandability, this system also supports up to 768MB of 100-MHz SDRAM DIMM memory.


Mid Level
 
Fathom N440BX
Specifically designed and tested for the server environment, this system's Intel N440BX server board delivers even greater throughput on memory and L2 cache access, as well as enhanced flexibility and expandability.
Odyssey N440BX-2
This server features hot-swap Ultra2 SCSI hard drive bays for increased uptime. Designed for robust cooling and added power protection, the chassis includes four independent fans.

Pelagos L440GX+
Featuring five Ultra SCSI hot-swap drive bays and two fans that support five 10k RPM hard drives, this system offers extreme reliability and ease of maintenance. Supports dual processors and up to 2 GB of SDRAM memory, with 4 DIMM sockets.

Navigator C440GX+

Enterprise Level
 
Encarta OCPRF Rack Mount Server
Featuring the Profusion chipset, this system supports up to eight Intel Pentium III Xeon processors, 32G of ECC memory, and advanced System Management. Processor scalability balanced with high I/O throughput.



Trident AC450NX Rack Mount Server
This server is designed for optimum use of rackmount space, and supports up to four Intel Pentium III Xeon processors. Delivers exceptional system reliability, availability and serviceability, with added cost value. Includes PCI hot plug functionality and advanced system management.

Cyclone AD450NX Server
This server features one to four Intel Pentium III Xeon processors for enhanced scalability and high-performance symmetric multiprocessing. Flexible features accommodate mid-range and high-end demands.

Leviathan SC450NX Server
Support from one to four Pentium III Xeon processors, 512 KB, 1 MB and 2 MB full speed L2 cache. Up to 4 GB of EDO system memory with 16 DIMM sites.
