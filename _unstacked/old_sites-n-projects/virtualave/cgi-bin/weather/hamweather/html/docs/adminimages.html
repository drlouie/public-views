Administration : Weather and Icons

HAMweather allows you to set the icons used when displaying the zone forecasts and the hourly and metar observations.  To modify these icon settings click the "Weather and Icons" link in the HAMweather administration options at the top left of any administration page. Before stepping through this admin it is best to discuss exactly how HAMweather deteremins what icons to use etc.

When HAMweather parses a zone forecast and creates the icon row it steps through the individual days forecast info.  It then runs down through a list fo weather conditions to look for, comparing an item in the list to the days foreacast info.  If it finds a match then it sets to use that weather conditions with the conditions icon and then moves on to the next day.  As you can see this is a first come first serve method. Because of this you should list  conditions such as "Partly Cloudy" and "Mostly Cloudy" before the condition "Cloudy" otherise HAMweather will output "cloudy" when it is actualy "Partly cloduy". (Because it found the word cloudy in Partly cloudy).  The default positioning of the various weather types should work sufficiently but you can move them around and insert new ones as needed.

When at the "Weather and Icons" screen, you will see a table of the various possible weather conditions with the associated day and night icons names.  The table will look similar to:

Icons Properties Link:
This link will allow you to set the height and width of the icons.

Day Titles Link:
This link allows you to set the display names/wording to use for the day titles in the icon row. For more information Goto end of page:

Show Icons Link:
This link will reload the page with the actual weather icons apearing in the day image and night image columns. Do note that this may cause the admin page to take longer to load in your web browser.

"D" Link (Delete):
This link is used to delete the current weather type.

"E" Link (Edit):
This link is used to edit the current weather type.

"I" Link (Insert):
This link is used to insert a new weather type before the current row.   When you click this you will see a form appear for you to enter the weather type, the display wording and the day and night image filenames.

Weather Type Column:
This is the weather type that HAmweather will look for in the actual forecasts.

Display Column:
This is the text HAMweather will use to when displaying the weather condition for the found weather type.  You can have fun here because you can set this to most anything you want.  This setting can also be set to a word in a foriegn   language to create a foriegn language version of HAMweather.

Day Image Column:
This is the filename of the image to use for the daytime image for the found weather type.

Night Image Column:
This is the filename of the image to use for the nighttime image for the found weather type.

Arrange Column:
This column is used to reposition a weather type up or down in the list.   Remember that weather types higher in the list are compared to the forecasts before the ones lower inthe list.  Simply click "up" to move the forecast type up one row or click "down" to move the forecast down one row.

NOTE:  These values are stored in the theme config file.  So when you make a change you are only making the change to the current theme for the current user.

Administration : Day Titles

HAMweather will allow you to edit the display names/text that is used for the actual day titles in the icon row. This allows you to control what is displayed and even easily make multilingual vaersions of HAMweather.  If a daytitle is needed that is not in the Table then HW will use the the actual word/phrase  given from the NWS.

Simply click the Day Titles link on the weather and icons admin page.  You will then see a table similar to:

Weather Info Link:
Takes you to the main Weather and Icons admin page.

Icon Properties Link:
Allows you to set the height and width of the weather icons.

Add New Link:
Allows you to enter a new day title and display name.

Del Link:
Allows you to delete the current day title row.

Edit Link:
Allows you to edit the current day title row.

IWIN Day Title Name Column:
This is the actual text given by the NWS/IWIN for the day title

Display Column:
This is the actual text to display for the given IWIN Day title word/phrase.
