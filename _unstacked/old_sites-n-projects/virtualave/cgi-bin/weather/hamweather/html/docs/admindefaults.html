Administration : Defaults and Options

Everything goes well when HAMweather is provided with all its needed parameters, but what would happen if HAMweather wasn't provided with these needed parameters?  To keep us from having to find out what would go wrong, HAMweather allows you to set defaults parameters.  These defaults allow HAMweather to properly handle requests that do not supply all needed parameters.  The defaults can be set by clicking the "Defaults and Options" link in the administration options at the top left of any administration page.

Following is a list of the various defaults and options with a description of each.

Default Place
This is the place that a forecast should be retrieved for  if not provided.

Default State
This is the state that a forecast should be retrieved for  if not provided.

Default Forecast Type
This is the forecast type HAMweather should use if a type isn't specified.   This can be a value from the list on the "Available Forecasts" page.

Default ICAO code
This is the 4 letter ICAO code to use as defaul when the Default forecast type is set to "metar".

Days Only Mode:
The days only mode tells what to display on the icon row for a zone forecast:

  • Normal : Display the weather icon row normally with day and night icons and use the day titles givien.
  • Display the icon row with days only (no nights) and use the normal day titles given
  • Display the icon row with days only (no nights) and use only weekday names (Monday, Tuesday etc)
  • Display the icon row with days only (no nights) and use only 3 letter weekday names (MON, TUE, etc)

Maximum Days
Set this with the value for the maximum number of icons to  create in the zone forecasts.

Extended Forecast
This controls whether HAMweather includes the extended forecast in the text or icons in the zone forecasts. 

Default Forecast Text Casing
This value is used to determine whether HAMweather leaves the forecast in upper case or performs Proper casing.

Text Mode
This value is used to determine whether HAMweather strips any html encoding from the text portion of a forecast.  Normally this would be set to 'No.'

Default Rounding:
This value is used to determine whether HAMweather rounds the Farenheit to Celsius conversions, Wind chill and heat index calculations.

Use Metric by Default?
This value is used to determine whether HAMweather uses english or metric units by default.

Default win unit when in metric mode
This value is used to determine what unit for wind HAMweather uses when in metric mode. (either KNots, MpS, or KM/H)

Day Begin Hour
This value is used to determine whether HAMweather should display a day or night icon for the current conditions. This shouldbe set to a number between 0 and 23. Where it is the hour that the sun rises.

Day End Hour
This value is used to determine whether HAMweather should display a day or night icon for the current conditions. This should be set to a number between 0 and 23. Where it is the hour that the sun sets.

Default Map Mode (Pro Only)
The map mode value HAMweather uses in case the map parameters is not provided when needed.   The settings for this value are the same as for the map parameter.

  • Set to '0' to stop maps from being created dynamically but to output the template.
  • Set to '1' to have map created dynamically and template outputted.
  • Set to '2' to have the map created dynamically  but do not output the template.

Allow Logging
This determines weather HAMweather will log all errors that occur as well as any requests that are made.  This is normally set to yes so that at minimum you'll have error logging.

User Template Path
This is not a default but an actual value.  This is the path to the users templates.  The checkbox to the left of this should ONLY be checked when you plan on changing this value.

Theme Images URL
This is not a default but an actual value.  This is the path to the URL to the directory that holds the weather icons used in the zone forecasts and hourly and metar observations..  The checkbox to the left of this should ONLY be checked when you plato on changing this value.
