Administration : Special Templates

HAMweather is a template based software package.  This is what allows HAMweather to be easily modified.  As seen in the Forecast Types page, a template name is given for each forecast type.  There also may be times when you need to output a forecast using an alternative template or there may be times when you need to output a template, but not a forecast.  Finally there also needs to be a template to display when HAmweather can't determine the forecast needed.

HAMweather allows for these possibilities in the names of Alt templates, pass templates and the not found template.  To administrate these templates click the "Special Templates" link in the administrations options at the top left of all administration pages.  The Special Templates admin screen will look similar to:

Alt Templates:
Alt templates are used when you need to output a forecast differently than the norm.  For instance, if you offer the zone forecast type to web users then you would use the normal template, but if you also allow a Java applet to display the text based zone forecast then you would want to use an alternate template that didn't have any HTML coding in it.

The alternate templates are stored in the users template directory and can be set from the "Special Templates" administration screen in the "alt templates" section.  When you view this admin the screen will look similar to image image above.

Nick Name Column: This is the nickname that you use in your urls and parameter strings.

Template Column: Thios is the actual template name associated with the Nick name.

Del Link: Click this link to delete the ALT template. NOTE: This does not actualy delete the html template from the hard drive but simply deletes the settings from HAMweather.

Edit Link: Click this link to edit the ALT template Nick name and.or actual template name.

Pass Templates:
Pass templates are used when you need to output a template/web page with out outputting a forecast. The pass templates help in allowing HAMweather to be a self contained software package.  An example of pass template usage in the HAMWEATHER demo is the output of the "Other National Info" page.  Here HAMweather outputs the other info.html template.  For instructions on usage of pass templates review the Accessing page.

The pass templates are stored in the users template directory and can be set from the "Special Templates" administration screen in the "pass templates" section.  When you view this admin the screen will look similar to image image ALT template admin screen except it will state & list the PASS templates.  The various attributes of the screen are the same as for the ALT templates.

Other Templates:
Other templates are template settigs that rewally do not fit anywhere else.  From here you can set the template to use when  no forecast is available.
