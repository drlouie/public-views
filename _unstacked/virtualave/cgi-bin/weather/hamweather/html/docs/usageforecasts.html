Usage:   Available Forecasts

Following is a table of the available weather information through HAMweather.  The generic label of forecast is used but actually some of the information available is actually reports, summaries and observations. 

In the table, the "forecast name" column contains the values that you will use for the "forecast" parameter when accessing HAMweather.  You can also click the forecast name for a demonstration of what it can output.(you will be taken to the  HAMweather website.)

In the description column you can read what the forecast type is and a general overview of what HAMWEATHER can output for this type. (See template customization for more information on outputting weather information.)

As of HW 2.0 Beta 3 you can now add your own additional forecast types using   information on the IWIN or from a URL of your choice.  Review the "Forecast Types" section of the Admin documentation for more information on how to set up additional forecast types.

Forecast Name Description
State Level Information
hourly Provides the hourly conditions for major cities in each state. This forecast is obtained from the IWIN system and isn't always available and is not available for all cities.

HAMWEATHER can output icon of current hourly weather, temp, wind, pressure, humidity.
HAMWEATHER Pro Add-on can also output  states regional radar image URL
zone Provides the zone forecast for a city, town or county. This forecast tends to be a three day forecasts.  In some cases the extended forecast is also available as part of the one forecast.

HAMWEATHER can output days row, icon row, condition & temp row, text forecast
HAMWEATHER Pro Add-on can also output  states regional radar image URL
zandh Provides both hourly and zone forecasts.  If the both hourly and zone forecasts are available for the place requested then sets of data can be outputted.  If the hourly conditions are not available then this will act just like the zone forecast.

HAMWEATHER will output data of both hourly and zone if available, otherwise it will act like zone or hourly depending on which forecast was available.
shortterm Provides the shortterm forecast for place when available.  This is usually a three to four hour forecast.

HAMWEATHER will can output the text of this forecast.
HAMWEATHER Pro Add-on can also output  states regional radar image URL
state Provides the forecast for the entire state. In some states it provides forecasts for the different regions.  NOTE: In versions 1.x, HAMWEATHER tried to up parse icons from the state forecast.  This has been abandoned in HAMWEATHER 2 do to the complexity of several state forecasts formats used by the IWIN system. So not HAMWEATHER will output the text of the state forecasts.

HAMWEATHER will can output the text of this forecast.
HAMWEATHER Pro Add-on can also output  states regional radar image URL
watches Provides the current active weather watches in a state

HAMWEATHER will can output the text of this forecast.
HAMWEATHER Pro Add-on can also output  states regional radar image URL
warnings Provides the current active weather warnings in a state

HAMWEATHER will can output the text of this forecast.
HAMWEATHER Pro Add-on can also output  states regional radar image URL
special Provides the current special weather statements (i.e. advisories etc) in a state

HAMWEATHER will can output the text of this forecast.
HAMWEATHER Pro Add-on can also output  states regional radar image URL
public Provides the general public information (if available) from the National Weather Service in a state

HAMWEATHER will can output the text of this forecast.
HAMWEATHER Pro Add-on can also output  states regional radar image URL
summary Provides the current weather summary (if available) for a state.

HAMWEATHER will can output the text of this forecast.
HAMWEATHER Pro Add-on can also output  states regional radar image URL
aviation Provides state wide aviation informaiton (if available) for a state.
discussion Provides a general discussion of weather info (if available) for a state.
National Weather Information
crop Provides the national crop summary (if available).

HAMWEATHER will can output the text of this forecast.
floodsummary Provides the national flood summary (if available).

HAMWEATHER will can output the text of this forecast.
earthquake Provides the report on major earthquakes and tsunamis that may occur (if available).

HAMWEATHER will can output the text of this forecast.
hurricane Provides the Report on current hurricanes that may threaten the US. (if available)

HAMWEATHER will can output the text of this forecast.
ultraviolet Provides the Report on forecasted Ultraviolet Index for the major cities in the US.

HAMWEATHER will can output the text of this forecast.
tornado Provides the current tornado warnings in the US.

HAMWEATHER will can output the text of this forecast.
thunderstorm Provides the current severe thunderstorm warnings in the US.

HAMWEATHER will can output the text of this forecast.
flashflood Provides the current flashflood warnings in the US.

HAMWEATHER will can output the text of this forecast.
flood Provides the current flood warnings in the US.

HAMWEATHER will can output the text of this forecast.
winterstorm Provides the current winterstorm warnings in the US.

HAMWEATHER will can output the text of this forecast.
marinewarning Provides the current tornado warnings in the US.

HAMWEATHER will can output the text of this forecast.
nonprecip Provides the national nonprecipitation summary(if available).

HAMWEATHER will can output the text of this forecast.
metar Provides the current metar observations (current conditions) at a specified weather facilitate through out the US.  you also need to specify the icao parameter with the wanted weather stations four digit ICAO identifier. 

HAMWEATHER can output icon of current hourly weather, temp, wind, pressure, humidity.
HAMWEATHER Pro Add-on can also output  states regional radar image URL
ns NOTE: Available with HAMWEATHER Pro Add-on Only.
Provides the current conditions of major predetermined cities through out the US. HAMweather will also create a weather map if you set the "map" parameter appropriately.

HAMWEATHER Pro Add-on can output text of current weather condition and temp of preselected cities. (Cities have to be listed in the IWIN's National Summary report).  Can also dynamically generate Current Conditions map.
nf NOTE: Available with HAMWEATHER Pro Add-on Only.
Provides the forecast of major predetermined cities through out the US. HAMweather will also create a weather map if you set the "map" parameter appropriately.

HAMWEATHER Pro Add-on can output text of forecasted weather condition and temp of preselected cities. (Cities have to be listed in the IWIN's Travelers  report).   Can also dynamically generate forecast map.
radar NOTE: Available with HAMWEATHER Pro Add-on Only.
Forces HAMWEATHER to generate a national radar map if needed.  Also HAMWEATHER will make regional radar maps as well.

HAMWEATHER Pro Add-on will generate the national and regional radar and display the national radar template as specified.
uswarnings NOTE: Available with HAMWEATHER Pro Add-on Only.
Provides a list of states that have currently active weather warnings. HAMweather will also create a national map that indicates which states currently have active warnings.

HAMWEATHER Pro Add-on can output text of the current states with warnings . (Cities have to be listed in the IWIN's Travelers  report).  Can also dynamically generate a US warning map.
International Weather Information
metar This is the exact same as metar under the National weather information. If an ICAO four digit identifier is provided for a weather facility throughout the world you can get the current conditions of that city.

HAMWEATHER can output icon of current hourly weather, temp, wind, pressure, humidity.
