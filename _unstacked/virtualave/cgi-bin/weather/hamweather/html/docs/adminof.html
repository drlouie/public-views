Administration : Output Files

HAMweather allows you to control the output of the forecasts.  The process for controlling this is specified in the HAMweather Output Usage page. You can control the output settings from the web based administration.  Each available setting is explained below:

Default Output Mode

This is the output mode that HAMweather uses when the 'om' parameter is not specified.   Normally, it should be set to zero (0) for web output only.  The other values that it can be set to are:

0 - Web only
1 - Web and File
2 - File only

Output File Path

This is the default directory path that HAMweather will ouput files to when needed.   To change this, simply modify the path in the textbox and then check the checkbox to the left.  Then when you submit the form the default Output Path will be chnaged. Note: NT users should include the drive letter in the path.

Default File Name

This is the default file name to use when outputting to a file and wht 'of' paramter was not specified.  To change this, simply modify the path in the textbox and then check the checkbox to the left.  Then when you submit the form the default file name will be chnaged.

Note: If you set the filename to include the full path then this value will override the default output file path value.

HAMweather Output Files

If you are going to be outputting to several files, then you should list them here.  

To add an output file, type in the output file nickname in a blank left most text box.  Then type in the filename of the actual file to be included in the rightmost text box.  Finally check the checkbox to the left of the include file nickname you added.  When you submit the form the output file will be added to the users output file list.   To remove an include file just unclick the checkbox to the left of the unwanted output file and submit the form.

Note: If you set the filename to include the full path then this value will override the default output file path value.
