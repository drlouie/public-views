Plans For HAMweather.

Well with the release of HAMweather 2.035, work is already being done on HAMweather 3.0. HAMweather 3.0 isa  complete rewrite of the HAMweather code.  With emphasis on resource usage you will find HW 3 to be smaller and much more efficient than HW 2.x   If all goes well there will be a beta product of HW3 in April 2000.

HAMweather Add-ons

Some additional add-ons for HAMweather are in the works.  The following should let you know what they are and help give an idea to their release:

RELEASED ADD-ONS:

  • HWscroll :  This will provide java applets that can be added to web pages to easily add forecasts etc.  RELEASED
  • HWzipdb: This is the zipcode database that can be added to HAMweather 2.0 Beeta 3.0 and higher to add zipcode searching to HAMweather. RELEASED
  • HWimage:  This is another feature similar to the bigger weather sites. This add-on will allow you to let web visitors add current condition images and forecast images to their web pages  as a single image.That when clicked will link back to the full forecast on your weather site. RELEASED
  • HWemail: This actually used to be part of the main HW distribution but has been seperated into its own add-on. RELEASED

ADD-ON IN THE WORKS:

  • HWpush : An active desktop add-on that can be used by web site visitors that have MS Win 98 and IE4+ users.  This will allow those users to add their forecast to their active desktop. So it can always be seen and it can be set to autoupdate.  Due out sometime during HAMwetaher 3 development.
  • HWmaps : An add-on that will allow you to create maps with weather conditions  for anyplace inthe world that HW can provide current conditions or forecasts.
